# Lab 1.1: ROS 2 Installation and Basic Publisher/Subscriber

## Objective
Install ROS 2 and create first publisher/subscriber nodes. Practice using ROS 2 command line tools and visualize data flow in RViz2.

## Prerequisites
- Linux-based operating system (Ubuntu 22.04 recommended)
- Basic understanding of terminal commands
- Minimum 4GB RAM and 20GB free disk space

## Learning Outcomes
- Successfully install ROS 2
- Create basic publisher and subscriber nodes
- Use ROS 2 command line tools for monitoring
- Visualize data in RViz2

## Lab Steps

### Step 1: ROS 2 Installation
1. Set up your locale
2. Add the ROS 2 apt repository
3. Install ROS 2 packages
4. Source the ROS 2 environment

### Step 2: Create a Workspace and Package
1. Create a new ROS 2 workspace
2. Create a new package for the lab
3. Set up the package structure

### Step 3: Implement Publisher Node
1. Create a publisher node that publishes a simple message
2. Configure the node to publish at a specific rate
3. Add appropriate logging and error handling

### Step 4: Implement Subscriber Node
1. Create a subscriber node that receives the published message
2. Process the received message and display it
3. Add appropriate logging and error handling

### Step 5: Build and Test
1. Build the package
2. Run the publisher and subscriber nodes
3. Monitor the communication using ROS 2 tools

### Step 6: Visualization in RViz2
1. Install and configure RViz2
2. Visualize the published data
3. Configure displays and panels

## Assessment
- Successfully install ROS 2 on your system
- Create working publisher/subscriber nodes
- Demonstrate communication using ROS 2 tools
- Visualize data in RViz2
