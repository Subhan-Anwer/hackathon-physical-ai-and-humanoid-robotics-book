# ROS 2 Fundamentals

## Learning Objectives
- Understand the differences between ROS 1 and ROS 2
- Learn about DDS (Data Distribution Service) concepts
- Implement nodes, topics, services, and actions
- Use parameter server and launch files

## 1. ROS 2 vs ROS 1 differences and improvements
- Architecture changes and improvements
- Middleware abstraction layer
- Improved security features
- Better real-time support

## 2. DDS (Data Distribution Service) concepts
- DDS architecture and its role in ROS 2
- Publishers and subscribers
- Data types and quality of service
- Discovery mechanisms

## 3. Nodes, topics, services, and actions
- Creating and managing nodes
- Topic-based communication patterns
- Service-based request/response communication
- Action-based goal-oriented communication

## 4. Parameter server and launch files
- Parameter management in ROS 2
- Using launch files to manage complex systems
- Parameter declaration and access patterns
- Launch file syntax and best practices

## Exercises
- Create a simple publisher and subscriber
- Implement a service client and server
- Use parameters in your nodes
- Create a launch file for a multi-node system
